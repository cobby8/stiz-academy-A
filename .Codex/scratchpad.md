- 2026-07-07: 관리자 공통 레이아웃을 모바일 메뉴 버튼/슬라이드 사이드바 구조로 조정하고 본문 모바일 여백을 최적화.

- 2026-07-07: 로그인 화면 다크모드에서 탭/입력창/제목 글자색이 배경에 묻히지 않도록 색상 대비 보강.
- 2026-07-07: 선생님 업로드 완료 상태 안내를 명확히 하고, 모바일 공개 메뉴/관리자 사이드바에 관리자·사진 업로드 진입점 추가.
- 2026-07-07: 선생님용 `/staff/quick-post`와 `SocialPostDraft` 초안 흐름, 관리자 승인 UI, 인스타 캐러셀 게시, PWA shortcut 추가.
- 2026-07-07: 인스타그램 자동 가져오기 cron을 매시간에서 하루 1회(한국시간 03:10)로 조정.
- 2026-07-07: 인스타그램 가져오기 공통 로직을 분리하고 수동 버튼과 cron이 같은 동기화 함수를 쓰도록 정리.
- 2026-07-06: 관리자 설정의 오래된 포토 갤러리 URL 입력을 제거하고 `/admin/gallery` 안내로 교체.

## 구현 기록
- 변경 파일: `src/app/admin/notices/NoticesAdminClient.tsx`, `src/app/actions/social-campaigns.ts`, `src/lib/instagram.ts`
- 주요 변경: 관리자는 공지사항 행에서 소셜 발행 모달을 열고, 공지 이미지로 인스타 피드/스토리 발행을 시도하거나 페이스북 광고 문구를 복사할 수 있다.
- 분리 기준: 공지사항 홍보 기록은 `SocialCampaignPost` raw SQL 테이블에 남기고, 공개 갤러리용 `GalleryPost`는 만들지 않는다.
