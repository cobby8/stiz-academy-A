# Decisions

## 2026-07-06: 프로젝트 정체성 확정
- 결정: 이 저장소는 STIZ 농구교실 다산점 홈페이지와 학원관리 플랫폼으로 본다.
- 이유: 실제 코드와 DB 모델이 상품/장바구니/주문형 쇼핑몰이 아니라 학원 운영 기능 중심이다.

## 기존 핵심 결정
- Next.js 16 App Router 기반으로 공개 페이지와 관리자 페이지를 함께 운영한다.
- Supabase PostgreSQL과 Prisma를 사용한다.
- Supabase PgBouncer 트랜잭션 모드 호환을 위해 raw SQL 패턴을 유지한다.
- 공개 페이지는 ISR 캐싱을 사용하고, 관리자/마이페이지는 실시간성이 필요한 곳에 동적 렌더링을 사용한다.
- Google Sheets CSV 기반 시간표 연동을 유지한다.
